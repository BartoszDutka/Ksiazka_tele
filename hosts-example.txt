# Przykład konfiguracji pliku hosts dla dostępu do Książki Telefonicznej

# =====================================
# GOTOWA KONFIGURACJA DLA TEGO SERWERA
# =====================================
# Adres IP serwera: 192.168.102.174
# Skopiuj poniższą linię do swojego pliku hosts:

192.168.102.174   igrabka.ksiazka-tel.pl

# =====================================
# INSTRUKCJA DLA WINDOWS
# =====================================
# 1. Otwórz Notatnik jako Administrator
# 2. Otwórz plik: C:\Windows\System32\drivers\etc\hosts
# 3. Dodaj powyższą linię na końcu pliku
# 4. Zapisz plik

# =====================================
# INSTRUKCJA DLA LINUX/MAC
# =====================================
# 1. Otwórz terminal
# 2. Wykonaj: sudo nano /etc/hosts
# 3. Dodaj powyższą linię na końcu pliku
# 4. Zapisz plik (Ctrl+X, Y, Enter w nano)

# =====================================
# DOSTĘP PO KONFIGURACJI
# =====================================
# Po dodaniu wpisu do pliku hosts aplikacja będzie dostępna pod adresami:
# 
# Główna strona:    http://igrabka.ksiazka-tel.pl
# Lista kontaktów:  http://igrabka.ksiazka-tel.pl/kontakty
# Logowanie admin:  http://igrabka.ksiazka-tel.pl/logowanie
# Dokumentacja API: http://igrabka.ksiazka-tel.pl/docs
